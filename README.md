## Install Dependencies

```
npm install
```

Node > 16.0.0 is needed.

## Run Development Environment

```
npm run dev
```

If the electron doesn't install completely, try to run this command:

```
node node_modules/electron/install.js
```

## Run Build

```
npm run build
```
